\begin{abstract}
Theory Exploration is a promising approach to improving the quality and understanding of software. It extends previously existing methods available through testing, in languages which are amenable to formal analysis such as those based on pure functional programming. Current theory exploration techniques are limited by their use of exponential time algorithms, which we mitigate by intelligently reducing the size of large inputs, using machine learning to preserve the relationships needed for successful exploration.
